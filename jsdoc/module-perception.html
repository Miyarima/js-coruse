Methods

(inner) clickedForm(el)

This function checks if the clicked form is the correct one, and if it is a point will be added
Parameters:
Name Type Description
el event the clicked form

(inner) drawForms()

This function is the main function of the perception test. It sets an interval for 20 seconds and inside a timeout for 1 second. each iteration a new form is displayed

(inner) finish()

This function displays the score of the test and changes the event on the procced button.

(inner) perception()

This functon set the title and descripton of the percption test. And if it's the first time loading, it will save the prior test score.

(inner) printFormText()

This function adds the description text for the test

(inner) randomForm()

This function draws a form depending on which iteration it's on

(inner) removeForms()

This function adds the hidden class to all elements with either the form-text or form class. And also removes the height of the btn-div.
